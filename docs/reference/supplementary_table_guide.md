# Supplementary-table guide

Our supplementary materials contain the primary result tables. We point readers to those tables rather than maintaining a second copy of the results.

| Table | Contents | Useful for |
|---|---|---|
| ST1 | Demographics and summary of 43 UK Biobank traits plus four All of Us replication traits | Phenotype definitions and sample sizes |
| ST2 | Ensembl gene IDs × 172 gene sets membership matrix | Reconstructing functional sets |
| ST3 | Jaccard overlap between all 172 gene-set pairs | Gene-set overlap analyses |
| ST4 | FunBurd estimates for 43 traits × 172 gene sets, including inside-set and outside-set effects for deletions and duplications | Primary association results |
| ST5 | Comparisons of significant association or enrichment proportions across deletions, duplications, and SNPs | Variant-class comparisons |
| ST6 | Brain-versus-non-brain comparisons across variant types and gene-set collections | Functional burden pleiotropy comparisons |
| ST7 | Brain-versus-non-brain comparisons across sensitivity analyses | Robustness checks |
| ST8 | All of Us replication FunBurd estimates for four traits × 172 gene sets | Cross-cohort replication |
| ST9 | Sign-concordance statistics between All of Us and UK Biobank | Replication summary |
| ST10 | Aggregated predicted loss-of-function burden estimates from GeneBass | Deletion–pLoF comparison |
| ST11 | Sex-stratified FunBurd estimates | Female and male analyses |
| ST12 | Sex-difference statistics for mental-health traits | Sex-aware interpretation |
| ST13 | S-LDSC functional enrichments across 172 gene sets | Common-variant comparison |
| ST14 | Gene-set-category comparisons across trait categories | Brain and non-brain specificity |
| ST15 | Constraint–pleiotropy correlations with overlap-aware p-values | Constraint analyses |
| ST16 | Gene-set-level pleiotropy, constraint, normative-null, CNV-targeting, and dosage-response metrics | Advanced functional interpretation |
| ST17 | Between-trait phenotypic, CNV-burden, pLoF-burden, and SNP-based correlations | Cross-trait architectures |
| ST18 | Cross-variant concordance statistics for between-trait correlations | Variant-class comparison |
| ST19 | Trait-level variance explained, deletion–duplication overlap, correlations, and dosage-response counts | Trait-level summaries |
| ST20 | BMI mediation analysis for selected correlated trait pairs | Direct versus mediated pleiotropy |
| ST21 | CNV length and protein-coding gene-content properties | CNV descriptive statistics |

## Column dictionary

The supplementary workbook includes an `ST_column_key` sheet describing the columns in each table.
