# FunBurd toy example

This folder contains the lightweight FunBurd example referenced by the documentation.

## Main files

- `FunBurd_Multitrait_EffectSizes_Computation_Script_On_ToyDatasets.ipynb`: readable notebook walkthrough
- `FunBurd_Multitrait_EffectSizes_Computation_Script_On_ToyDatasets.py`: command-line script
- `Args_File_FunBurd.txt`: example commands spanning toy phenotypes and three gene-set collections
- `Genotypic_info_Toy/`: toy CNV intervals and CNV-to-gene annotations
- `Phenotypic_info_Toy/`: ancestry and toy phenotype files
- `Gene_Sets/`: example tissue and cell-type gene sets
- `Outcome_Toy/`: sample output tables

The example is educational and intentionally lightweight. Publication-scale analyses and figure-generation workflows remain in the main GitHub repository and archived release.
